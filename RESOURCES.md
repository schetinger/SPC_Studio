# Agent Skills Resources

## Knowledge

- [skills.sh — mattpocock/skills](https://skills.sh/mattpocock/skills)
  Página oficial do pacote de skills. Lista todas as skills disponíveis com descrições. Use para: descobrir o propósito de cada skill antes de invocar.

- [Repositório GitHub: mattpocock/skills](https://github.com/mattpocock/skills)
  Código-fonte de todas as skills. Use para: ler o `SKILL.md` de cada skill e entender exatamente o que o agente vai fazer.

- [Antigravity IDE — documentação interna das skills instaladas](./.agents/skills/)
  Skills instaladas localmente neste repo. Use para: verificar a versão exata instalada e ler instruções offline.
