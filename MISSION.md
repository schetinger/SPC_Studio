# Mission: Agent Skills (mattpocock/skills)

## Why

Delegar partes do desenvolvimento do projeto CEP (Django) para agentes orientados a spec — usando as skills já instaladas — de forma a acelerar o ciclo de desenvolvimento: especificar → revisar → iterar, sem precisar fazer tudo manualmente.

## Success looks like

- Conseguir criar uma issue bem-especificada no GitHub a partir de uma ideia usando `to-issues`
- Conseguir rodar `tdd` para implementar uma feature com loop red-green-refactor guiado pelo agente
- Conseguir usar `review` para auditar um PR ou branch contra a spec original
- Saber quando chamar cada skill e o que esperar como output

## Constraints

- Foco exclusivo no projeto CEP (Django): `app/`, `models.py`, `views.py`, `utils.py`
- Aprende por uso prático — não quer teoria abstrata desconectada do projeto real
- Quer dominar o uso das skills existentes antes de criar novas

## Out of scope

- Eletrônica (o projeto tem esse nome mas não é o foco agora)
- Criar skills novas (`write-a-skill`) — próxima etapa
- Skills de facilitação pessoal (`grill-me` puro, `teach`) como tópico em si
