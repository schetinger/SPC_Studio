# Mission: SPC Studio

## Why
Prover uma plataforma web robusta e educacional para **Controle Estatístico de Processos (CEP)**. Ela serve para processar amostras e gerar relatórios matemáticos com Cartas de Controle padronizadas. Ao mesmo tempo, atua como um sistema de ingestão IoT (sinal real) para provar a utilidade do motor estatístico com hardware rodando na ponta em tempo real.

## Success looks like
- Motor Estatístico perfeitamente isolado e testado por TDD, processando com rigor Cartas X-R, P, I-MR, U etc.
- A API de ingestão REST rodando estável e respondendo instantaneamente no frontend.
- Um design responsivo (Mobile First puro no CSS) e uma UI moderna e bonita.

## Constraints
- Foco em resolver problemas com simplicidade: uso de HTTP Polling, CSS Vanilla, etc.
- O IoT é acessório da plataforma, o coração da plataforma é a Engenharia e a Estatística.
